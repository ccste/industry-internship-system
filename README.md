# 🎓 Industry Internship System (IIS)

A multidisciplinary internship project demonstrating my skills in **Software Testing**, **Data Analysis**, and **UX/UI Design** through a real-world education–industry workflow system.

<br>

## 📌 Project Overview

The **Industry Internship System (IIS)** is a web-based application designed to streamline the internship placement process for:

- Wintec IT students  
- Local Hamilton businesses  
- The Industry Relationship Manager (IRM), Sue Pearson  

Before this project, the internship placement workflow relied heavily on **Google Forms**, **manual review**, and **Canva Board updates**, which created inefficiencies in data handling, communication, and administrative work.

The goal of IIS was to build a centralized system where:

- Students could submit and manage internship applications  
- The IRM could review and publish student profiles more efficiently  
- Industry clients could browse candidate information and arrange interviews directly  

This repository showcases the parts of the project I completed during my **16-week internship**, with a strong focus on:

- 🧪 **API Testing (Primary Focus)**  
- 📊 **Power BI Data Analysis**  
- 🎨 **UX/UI Design**
